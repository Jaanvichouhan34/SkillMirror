## 🪞 What is SkillMirror?

SkillMirror is an **AI-powered student performance evaluator** built for CS students preparing for top-tier placements. It doesn't just test you — it diagnoses you. 600+ questions across 6 domains, 10 progressive levels each, and a full gamification layer to keep you coming back.

## ✨ Features

### 🎯 Assessment Engine
- **600+ unique questions** across 6 CS domains
- **10 progressive levels** per domain — beginner to placement-ready
- **80% pass threshold** with detailed per-question explanations
- **Perception Gap Analysis** — shows where you *think* you stand vs where you actually do

### 📚 Learning Paths
- **7 structured paths** — Python, JavaScript, C++, Web Dev, SQL, Java, C Systems
- **120+ modules** per path — theory, code examples, and embedded quizzes
- Interactive code playground for hands-on practice
- Progress tracking per module and path

### 🏆 Gamification System
- **XP & leveling** — earn points for every quiz, lesson, and streak
- **20+ achievement badges** with unlock conditions
- **Daily streaks** — maintain your momentum
- **Skill diagnostic reports** — shareable, visual, interview-ready

### 🎨 Premium UI/UX
- 🌌 Deep Space Dark Theme `#04050a` + Light Mode toggle
- ✨ Glassmorphism cards with custom cursor animations
- 🎵 Ambient music player with equalizer visualizer
- ⚡ Framer Motion page transitions throughout
- 📱 Mobile-first, fully responsive

## 🚀 Getting Started

### Prerequisites

- Node.js `16+`
- npm or yarn

### Quick Setup

```bash
# Clone & install
git clone https://github.com/Jaanvichouhan34/SkillMirror.git
cd SkillMirror
npm install

# Development
npm run dev

# Production build
npm run build
npm run preview
```
